A simple React movie browser application that allows users to search for movies and view results fetched from the TMDB API.

## Features
- Search movies by title
- Fetches real-time movie data from the TMDB API
- Displays movie search results dynamically
- Clean and simple UI

## Tech Stack
- React
- JavaScript (ES6)
- HTML & CSS
- TMDB API

## Getting Started

Install dependencies:
npm install

Create a .env file in the root directory and add your TMDB API key:
REACT_APP_TMDB_API_KEY=your_api_key_here

Start the development server:
npm start

The app runs at:
http://localhost:3000

## Notes
- A TMDB API key is required to run this project.
- The .env file is ignored by Git.
- This project is for learning and practice purposes.
